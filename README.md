# sports-project

This project explores different datasets across various of different sports industries. The end goal for this project is to create a "Hub" for sports analytics for different sports industries & leagues.

<Process of this project>
1. Extracts dataset in csv file using pipeline through Python 
2. Pushes extracted dataset to database through SQL query
3. Perform data analytics and machinle learning processs through R and Python
4. Publish the results out through html
